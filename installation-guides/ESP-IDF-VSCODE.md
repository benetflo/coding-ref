- Open VS Code -> Extensions -> Install ESP-IDF extension.
- Open the newly installed extension in the bar on the left side.
- Click advanced -> Configure ESP-IDF extension -> Express
- In the newly opened window, choose latest ESP-IDF version and leave everything else as default.
- Click Install.
- Create a new project via quick actions or "New Project Wizard"
- Press Enter to choose ESP framework.
